## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:  
   `npm install`
2. Create a `.env` file in the project root (this file should **not** be committed to Git) and set your NASA API key:  
   `VITE_NASA_API_KEY=your_nasa_api_key_here`
3. Run the app:  
   `npm run dev`
